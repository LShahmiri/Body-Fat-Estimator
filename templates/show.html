About This Estimation

This web application is based on the Body Fat Dataset, which contains anthropometric measurements collected from adult male subjects. The dataset includes body density values measured through underwater weighing together with several body circumference variables such as chest, abdomen and hip.

Body fat percentage can be derived from body density using Siri's equation (1956):

Body Fat % = 495 / Density − 450

In this project, a machine learning regression model was trained to estimate body fat percentage using selected body measurements that are easy to obtain in real-world health or fitness environments.

The dataset was originally compiled and distributed by:

This project uses the dataset for educational and demonstration purposes. The prediction results should not be considered a medical diagnosis.
